# 배포 전 체크리스트 (필수)

## 1) 로컬 빌드/린트
- `npm run predeploy`

## 2) AI 분석 화면 스모크
- `/works/[id]/analysis` 진입
  - 개별 분석 리스트에서 **종합 점수/플랫폼이 `—`가 아닌지**
  - 특정 회차 `개별 분석` 클릭 시 **최신 리포트가 자동으로 표시되는지**

## 3) 글자 수(원고량) 관련
- 개별 분석 패널에서 글자 수가 오래 “계산 중…”으로만 남지 않는지
  - 실패하면 “불러오기 실패”로 전환되는지
- 통합 분석에서 **글자 수를 아직 모르는 회차(0)** 때문에
  - “500자 미만” 경고가 **오탐으로 뜨지 않는지**
