# OneHandBook (OHB)

> "내 작품이 묻히는 게 아니라, 발견되지 않은 것이다."

웹소설 플랫폼의 알고리즘 한계를 넘어, 신인 작가와 독자가 직접 연결되는 Web3.0 기반 창작 생태계입니다.

---

## 왜 만들었냐

웹소설 작가로 살아봤다.
글을 올리면 한 시간 뒤에 피드에서 사라진다.
조회수가 없으면 내 글이 나쁜 건지, 아무도 못 본 건지 알 수가 없다.
플랫폼은 이미 흥행한 작품만 밀어준다.
**신인 작가는 독자를 만날 기회조차 없다.**

OneHandBook은 그 구조를 뒤집는다.

---

## 뭘 하는 거냐

| 기존 플랫폼 | OneHandBook |
|------------|-------------|
| 흥행한 작품만 노출 | 흥행 **가능성**을 먼저 점수로 산출 |
| 독자는 소비만 | 독자가 초기 작품에 **투자** |
| 작가 수익 45~55% | **90% 이상** 작가에게 귀속 |
| 플랫폼 의존 | **Direct-to-Fan** 직접 연결 |

---

## 핵심 기능

### Oracle Tomato — 흥행 지수

- 체류시간, 정주행률, 이탈 구간을 분석
- 0~100점으로 흥행 가능성 산출
- **조작 불가능한 숫자**

### 작가 대시보드

- 몇 화에서 독자가 빠지는지 시각화
- 플랫폼이 절대 알려주지 않는 인사이트 제공

### Direct-to-Fan

- 수익의 90% 이상이 작가에게
- 카카오, 네이버의 45% 수수료 구조를 거부
